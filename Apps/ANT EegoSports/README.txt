This directory contains the labstreaminglayer application for the ANT eego sports amplifier. 

The labstreaminglayer has originally been developed by the Swartz Center for Computational Neuroscience in San Diego. This application has been developed by the Laboratory of Brain-Computer Interfaces at Graz, University of Technology. 


The executable can be found in Win32/Release/eegoSports.exe. Make sure that all necessary drivers are installed and that a ANT license file is pressent at [User]/My Documents/eego. See the amps API documentation for further details. 

This application uses the eego-SDK 1.3.4.
